# GugleCarpetAddition NeoForge

GugleCarpetAddition for NeoForge. Ported from Fabric to NeoForge.

## About

This project is the NeoForge port of the **GugleCarpetAddition (GCA)** mod.

- **Original Author**: Gu-ZT
- **NeoForge Port**: AlkSur
- **Original Project**: [GugleCarpetAddition](https://github.com/Gu-ZT/gugle-carpet-addition)

## Dependencies

- **Carpet: NeoForged** (version `neoforge-carpet-1.21.1-1.0.8+v251027` or compatible)

## Features

GugleCarpetAddition is an extension to the Carpet mod, providing additional gameplay improvements and debugging tools. For detailed features, please refer to the original project documentation.

## Installation

1. Ensure NeoForge and Carpet: NeoForged are installed
2. Place this mod's jar file in the `mods/` directory
3. Launch the game
