# GugleCarpetAddition NeoForge

GugleCarpetAddition 的 NeoForge 移植版本。从 Fabric 移植到 NeoForge。

## 关于

本项目是 **GugleCarpetAddition (GCA)** 模组的 NeoForge 移植版本。

- **原作者**: Gu-ZT
- **NeoForge 移植**: AlkSur
- **原项目**: [GugleCarpetAddition](https://github.com/Gu-ZT/gugle-carpet-addition)

## 前置模组

- **Carpet: NeoForged** (版本 ≥ 1.4.147)

## 功能特性

GugleCarpetAddition 是 Carpet 模组的扩展，提供额外的游戏性改进和调试工具。具体功能请参考原项目文档。

## 安装

1. 确保已安装 NeoForge 和 Carpet: NeoForged
2. 将本模组的 jar 文件放入 `mods/` 目录
3. 启动游戏
